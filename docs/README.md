# Project Documentation Index

`docs/` 文档按用途组织为产品规格、设计文档、执行计划、生成产物和参考资料。

## Directory Outline

- `product-specs/`: 产品规格、能力范围和产品层规划。
- `design-docs/`: 架构设计、技术设计和实现蓝图。
- `exec-plans/`: 执行计划目录。
  - `active/`: 仍在推进或后续需要继续维护的执行计划。
  - `completed/`: 已完成的执行计划和实施记录。
- `generated/`: 自动生成或阶段性汇总产物。
- `references/`: 调研笔记、外部资料摘要和历史参考资料。
